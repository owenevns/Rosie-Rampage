# Rosie-Rampage
<img width="752" height="658" alt="image" src="https://github.com/user-attachments/assets/405b4e94-9cae-4e86-b0a5-cfa3dcba8a9a" />

150g flyweight battle bot with spinner weapon. This was designed for the WatBots battle bots club.

It features a fully machined aluminum "eggbeater" weapon attached to a 6A drone motor. 

We made a custom PCB to interface the 3 motor ESCs, battery and ESP32 supermini c3 board.

The controls were done using a dualshock 4 controller; communicating over bluetooth using the bluepad32 library and an RF antenna trace.
